# Student Performance Analytics Dashboard

## Project Overview

This project focuses on analyzing the academic performance of students using Python and Power BI. The objective was to study different factors affecting student performance and extract meaningful insights from the available data.

The analysis was carried out on semester-wise SPI data along with attendance, gender, and accommodation details. Various visualizations and an interactive dashboard were created to understand trends and patterns in student performance.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI
* Jupyter Notebook

---

## Dataset Information

The dataset contains information about students including:

* Student Name
* Roll Number
* Gender
* Attendance Percentage
* Accommodation Type
* Semester-wise SPI

Missing SPI values were treated as due credits, which helped in performing backlog analysis.

---

## Analysis Performed

The following analyses were carried out during the project:

* Attendance Analysis
* Gender-wise Performance Analysis
* Accommodation-wise Analysis
* Semester-wise SPI Distribution
* Performance Category Analysis
* SPI Change Analysis
* Backlog Analysis
* Correlation Analysis

---

## Key Insights

* Students with higher attendance generally showed better academic performance.
* Most students belonged to the Good and Average performance categories.
* Female students had slightly higher average SPI.
* Hostellers showed marginally better performance than day scholars.
* A significant number of due credits were observed in the fourth semester.
* Semester-wise comparison helped identify students whose performance improved or declined.

---

## Power BI Dashboard

An interactive dashboard was developed to provide a consolidated view of the analysis. The dashboard includes:

* KPI Cards
* Attendance vs SPI Analysis
* Gender-wise SPI Comparison
* Accommodation-wise SPI Analysis
* Performance Category Distribution
* SPI Change Analysis
* Interactive Filters and Slicers
